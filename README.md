# Coffee Shop Landing Page

Build a modern landing page for Sunrise Coffee Co. Features: hero section, menu display, location map, online ordering, mobile responsive. Include AI chatbot.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Built With

- Next.js 14
- TypeScript
- Tailwind CSS
